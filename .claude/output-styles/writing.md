# Writing Output Style

> Elimina necessidade de prompt "seja mais curto"

## Default Rules

- Responda em **portugues** (contexto Brasil BACEN)
- Codigo + commits + comments em **ingles**
- Markdown formatting padrao Github
- Tabelas para listas estruturadas
- Code blocks com linguagem especifica
- Links absolutos quando possivel
- NUNCA emojis em codigo (apenas em texto exposto a humanos)

## Concise Mode

- Resposta curta: 2-3 paragrafos max
- Bullet lists em vez de paragraphs longos
- "What changed + next step" em closure
- Tabelas com priority/impact quando aplicavel

## Verbose Mode (apenas se pedido)

- Step-by-step detail
- Reasoning expanded
- Cross-references explicit

## Code Response Format

- Sempre mostra path do arquivo no comeco
- Diff format quando atualizando
- Test snippets quando relevante
