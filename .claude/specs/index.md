## Convention

- 1 spec per major feature
- Status workflow: DRAFT → REVIEW → APPROVED → IMPLEMENTED
- Cross-reference milestone (MS-XXX) + RN_FX rules

## When to write a spec

- Feature toca > 3 BCs
- Feature tem business value claro mas implementation unclear
- Feature requires alignment cross-team
- Feature requires regulatory review
